Pattern

Pattern allows you to use regular expressions to determine access.

Specification
Result
Pattern("printers.edit",
        PatternType.EQUALITY,
        new MyDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
Pattern("printers.foo",
        PatternType.EQUALITY,
        new MyDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
Pattern("(.)*\\.edit",
        PatternType.REGEX,
        new MyDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
