SubjectPresent

This is one of the simplest constraints in Deadbolt. It simply requires that a user be present (i.e. logged in).

Specification
Result
SubjectPresent(new MyDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
SubjectPresent(new MyUserlessDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
