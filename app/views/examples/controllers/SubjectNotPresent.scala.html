SubjectNotPresent

The counterpart to SubjectPresent. This constraint requires that a user isn't present (i.e. no-one is logged in).

Specification
Result
SubjectNotPresent(new MyDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
SubjectNotPresent(new MyUserlessDeadboltHandler) {
  Action {
    Ok(accessOk())
  }
}
